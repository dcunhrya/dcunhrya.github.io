Circuit Design/PCB

Circuit designed in KiCad with main functionality being voltage regulating circuit and Arduino inputing sensor readings and powering LEDs. Prototype on breadboard before designing double sided PCB in KiCad. Components soldered to board with each individual circuit track tested.

Microcontroller

Arduino Nano 33 BLE used to connect device with app via BLE. Currently coding firmware in BLE functionality and optimizing sleep modes along with interrupts. Project built in PlatformIO using nRFConnect with plans in future to use Zephyr for real-time operating system functionality

CAD Enclosure

A CAD enclosure was designed in OnShape and printed in PLA. It contains a housing to mount the PCB in as well as external connections for the LEDs and replaceable batteries. A universal joint was created to make a movable tube the user can breathe into

Software

There is an Android and Apple breathalyzer app coded in Dart and Swift respectively. Both apps used Figma to create a mock UI/UX and have indentical functionality. Apps have a display screen to view BAC readings and a library to log and track drinks consumed

Hardware Components

The main hardware components used were a fuse, CR2032 batteries, a low dropout voltage regulator (MAX6105), and a MQ3 sensor. There is also a barrel jack coupled with the fuse in order to power and heat up the sensor prior to use in order to avoid draining batteries.

MQ3 Sensor

Metal oxide semiconductor containing an AlO3 ceramic coated with SnO2. The AlO3 and Ni-Cr coil handle the sensor heating and the SnO2 measures the alcohol. When SnO2 layer is heated, oxygen adsorbs to the surface. In clean air, electrons on the SnO2 layer attract to the oxygen creating a resistive barrier preventing current flow. Adsorbed oxygen reacts with alcohol if present. Electrons are then released to the SnO2 layer and current flows freely resulting in a higher voltage output.
