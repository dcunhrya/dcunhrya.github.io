[December 2022 - January 2023]

Arduino-based device measuring SpO2 (oxygen saturation) and heart rate using MAX30102 sensor and displaying data on OLED display

  • Skills: Microcontrollers, C, Circuit Design, OnShape (CAD)

Hardware Requirements

    • Arduino Mega
    • OLED Display (SSD1306)
    • MAX30102 Sensor

Software Dependencies

    • Arduino IDE
    • Adafruit SSD1306 library
    • Adafruit GFX library

Features

Scrolling Image Gallery:

Circuit

A MAX30102 Heart Rate sensor, 128x64 OLED display, and an Arduino Mega were used. MAX30102 contains an IR LED, red LED, and photodiode in order to measure SpO2 from oxygenated and unoxygenated blood from the tip of the finger. Both the MAX30102 and OLED were connected using I2C

CAD Casing

A CAD casing was printed in PLA for the OLED display, heart rate sensor, Arduino top case, and Arduino bottom case. 4 separate prints were all combined to integrate the device. All dimension drawings are shown in the image gallery and available on the GitHub

Code Integration

All code for the pulse oximeter was written in the Arduino IDE in C/C++. Code used several Arduino libraries containing algorithms to calculate SpO2 levels and heart rate. Filters were used in the libraries to remove noise, and code was optimized to use less storage space and dynamic memory. Code is shown on GitHub.
